# Dashboard Estoque

> Sistema de Dashboard para Controle de Estoque

## 🚀 Tecnologias

Este projeto foi desenvolvido com as seguintes tecnologias:

- **Frontend:** Vue 3, TypeScript, Vite, Vuetify, Pinia, Axios, Apex Charts 
- **Backend:** Node.js, TypeScript, Express
- **Banco de Dados:** Firebird 

---

## 🔧 Como Executar o Projeto

> ⚠️ **Nota de Execução:** Este projeto foi desenvolvido sob uma estrutura e modelagem de banco de dados altamente específica para o cenário da fábrica parceira. Devido a essas dependências de dados e consultas personalizadas, **o projeto não está configurado para execução local imediata**.

O objetivo deste repositório é servir como **portfólio técnico**, permitindo a avaliação da arquitetura do código, boas práticas de desenvolvimento e a lógica de negócios aplicada.
